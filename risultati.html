Progetto

questo sito e' stato realizzato con amazon web services e i file sono stati caricati tramite l'istanza del server ftp.

gli sfondi sono stati presi da internet e le credenziali del login sono state asportate dall'istanza di mariaDB.

a livello piu' tecnico il codice per creare questo sito e':

sudo apt update

sudo apt upgrade

sudo apt install docker.io

sudo service docker status

per il server ftp:

sudo docker run -d --name=ftpserver --network=host -v /home:/home/vsftpd -e FTP_USER=dany -e FTP_PASS=123456 -e PASV_ADDRESS=127.0.0.1 -e PASV_MIN_PORT=15000 -e PASV_MAX_PORT=15100 --restart unless-stopped fauria/vsftpd

per il mariaDB:

sudo docker run -d --name=mariadb -e PUID=1000 -e PGID=1000 -e MYSQL_ROOT_PASSWORD=123456 -e TZ=Europe/Rome -e MYSQL_DATABASE=isisPonti -e MYSQL_USER=johnsmith -e MYSQL_PASSWORD=123456 -p 3306:3306 -v mysql_data:/config --restart unless-stopped linuxserver/mariadb

per l'auto certificazione:

sudo openssl req -x509 -nodes -days 365 -newkey rsa:2048 -keyout ~/c_progetto.pem -out~/cert.pem

i file sono stati caricati tramite filezilla
